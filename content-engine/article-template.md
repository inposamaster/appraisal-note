# 글 표준 템플릿

`content-plan.csv`의 `suggested_title`을 H1으로 사용한다.  
제목 공식: **[핵심 키워드] + [상황] + [궁금증/해결]**

품질·페르소나: `quality-standard.md`  
H2는 **고정 1~5가 아니라** `content_type`에 맞게 4~6개를 고른다.

## 공통 본문 골격

모든 글에 공통:

1. **두괄식 서론** (2~3문단) — 첫 문장에 결론
2. **핵심 요약** (인용 블록 3~5줄)
3. **H2 본문** 4~6개 (`content_type`별 아래 참고)
4. **시각 요소** — `visual-elements.md` 참고
   - 표 **1~2개**, 핵심 요약 박스 **1개**, 주의 박스 **0~1개**
   - 절차·분기: **Mermaid** 1개 (발행 전 PNG로 교체 가능)
5. **FAQ 3개** — 본문 근거 + 롱테일
6. **마무리** + YMYL 고지문

인사말·특정 사무소명 금지.  
「감정평가란」 기초 설명은 2문장 이내 → [감정평가 기초 소개](/appraisal-note/posts/what-is-real-estate-appraisal/) 링크.

## content_type별 H2 가이드

### 비교정리 (`search_intent`: 비교형)

- {핵심 선택지 A vs B} — 무엇이 다른가
- {이런 경우 A가 맞을 수 있다}
- {이런 경우 B를 검토할 수 있다}
- {판단 전 확인할 점 / 흔한 오해}
- (선택) {다음에 읽을 주제}

→ **비교 표** 필수

### 개념설명 (`search_intent`: 정보형)

- {왜 이 질문이 생기는가}
- {제도·용어 정리 — 짧게}
- {이런 상황에서 검토가 필요할 수 있다}
- {놓치기 쉬운 점}
- (선택) {관련 절차로 이어지는 한 줄}

### 절차안내·체크리스트 (`search_intent`: 실행형)

- {전체 흐름 한눈에}
- {단계별로 무엇을 하는가}
- {준비 서류 — 이 글에 필요한 것만}
- {흔한 실수}
- (선택) {의뢰·신고 전 최종 확인}

→ **번호 목록 또는 체크리스트 표**

### 사례해설

- {상황 설정 — 일반화, 가명 인물 금지}
- {어디서 막히는가}
- {감정평가·신고 관점에서 볼 점}
- {유사 상황 체크}

## 준비서류 표 (필요할 때만)

| 자료 | 확인 목적 |
|---|---|
| 등기사항전부증명서 | 소유자, 권리관계 |
| 건축물대장 | 용도, 구조, 면적 |
| 토지대장 | 지목, 면적 |
| 토지이용계획확인서 | 용도지역, 규제 |
| 임대차계약서 | 임대수익, 보증금 (해당 시) |

**매 글 풀세트 복붙 금지.** 해당 글에 필요한 행만.
