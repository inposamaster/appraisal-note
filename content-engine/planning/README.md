# 키워드 기획 (0단계)

파이프라인 **0단계** 산출물 폴더. 생산(1~7) 전에 여기서 끝낸다.

**SSOT:** 발행 큐는 항상 `../content-plan.csv`. 이 폴더는 **기획 근거·후보**만.

---

## 파일 역할

| 파일 | 역할 |
|------|------|
| `keyword-pool.csv` | 후보 키워드·질적 점수·중복 메모·5개월차 풀 |
| `month-*-keyword-sprint-*.md` | 스프린트별 **최종 12편** 선정 근거·주간 순서·제외 사유 |
| (없음) | 스프린트마다 md 1개 추가. 완료 후 아카이브 가능 |

---

## 스프린트 → CSV 반영 절차

1. 후보 풀 작성·점수 (`keyword-pool.csv`)
2. 1~N편 기발행·다른 블로그와 **중복 스캔**
3. 최종 N편·주간 순서를 스프린트 md에 기록
4. `content-plan.csv`에 행 추가 (`validation_status: 검증완료`, `status: planned`)
5. `seed-priority.csv` 갱신 (새 시드 소진 시)
