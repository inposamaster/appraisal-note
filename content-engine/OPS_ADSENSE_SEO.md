## 계정 안전 수칙 (04-03 · 필수)

의지력만 믿지 말고 **환경·게이트**로 막는다. 정지는 예방이 전부다.

| 규칙 | 실천 |
|------|------|
| **자가클릭 금지** | 본인·가족·지인에게 「광고 눌러달라」 **절대 금지**. 점검은 **클릭 없이** 육안만. 가능하면 광고차단 ON인 **점검용 브라우저** 분리 |
| **클릭 유도 금지** | 「광고 눌러주시면…」「여기를 보세요」화살표·광고 위장 **템플릿·본문 어디에도 없음** |
| **정직한 트래픽만** | 유료 방문자·자동 클릭·봇 구매 **금지**. 검색·공유만 |
| **검수 게이트** | 자동 생성 = `draft: true` → 사람 검수 후만 `false`. AI가 책임까지 맡지 않음 |
| **발행 리듬** | **하루 1~2편·주 수편** 맥박. 하루 수십 편 폭발 양산 **금지**. 시각은 기계적 00:00 고정 지양 |
| **얇은 복제 금지** | 날짜·숫자만 바꾼 동일 본문 금지. 검수 때 「어제와 사실상 같은가」 |
| **월 1회 정책 센터** | 매달 1일(권장) AdSense **정책 센터** 경고 확인. 세부는 **구글 공식 정책**만 |
| **광고 코드 단일화** | head partial + `hugo.toml` params만. 테마 손볼 때 스니펫 **중복·오타** 주의 |

**검수 게이트 3초 훑기:** ① 사실·숫자 ② 복제·양산 느낌 ③ 단정·정책 위험 표현.
