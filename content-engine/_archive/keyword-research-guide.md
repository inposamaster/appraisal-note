# 키워드 발굴·검증 가이드

## 기본 방향

- **머리 키워드가 아니라 롱테일 키워드**를 노린다.
- 검색량이 아주 큰 키워드보다 **검색 의도가 분명한 키워드**를 우선한다.
- AI가 제안한 키워드는 실제 검색 데이터가 아니라 **후보일 뿐**이다.

## 검증 도구 (사람이 직접 확인)

| 도구 | 확인 내용 |
|---|---|
| Google 자동완성 | 실제 검색자가 입력하는 연장 키워드 |
| Google 관련 검색어 | 검색 결과 하단 연관 키워드 |
| Google Keyword Planner | 검색량·경쟁도 참고 (단, 숫자만 믿지 않는다) |
| 네이버 데이터랩 | 국내 검색 트렌드·연관어 참고 |

검증 결과는 `keywords.csv`의 `source_hint`, `competition_note`, `validation_status`에 기록한다.

## 경쟁도 판단 기준

경쟁도는 **광고 경쟁도가 아니라 실제 검색 결과 1페이지**를 보고 판단한다.

### 후순위로 둘 키워드

- 대형 언론사, 정부기관, 대형 플랫폼만 상위에 있는 경우
- 검색 의도와 맞지 않는 결과가 많은 경우

### 진입 가능성이 있는 키워드

- 개인 블로그, Q&A, 카페 글이 1페이지에 섞여 있는 경우
- 질문에 대한 직접적인 답변이 부족한 경우
- 최신 정보가 오래된 경우

`competition_note`에는 "1페이지에 ○○ 유형 사이트 다수"처럼 **관찰 내용**을 적는다. 숫자로 단정하지 않는다.

## CPC(광고 단가) 메모

- `cpc_note`에는 "광고 단가가 받쳐줄 가능성"에 대한 **질적 메모**만 적는다.
- 예: "금융·세금 연계 키워드로 단가 가능성 있음 (미확인)"
- AI나 도구가 제시한 숫자를 그대로 기록하지 않는다.

## 검색 의도 분류

| 유형 | 설명 | 예시 |
|---|---|---|
| 정보형 | 개념·차이·필요 여부를 알고 싶다 | "공시지가와 감정평가액 차이" |
| 비교형 | A와 B를 비교하고 싶다 | "증여 부동산 감정평가 비용" |
| 실행형 | 방법·절차를 알고 싶다 | "경매 감정평가서 보는 법" |

## 콘텐츠 유형 분류

| 유형 | 적합한 주제 |
|---|---|
| 개념설명 | "~란?", "~이 필요한 경우" |
| 절차안내 | "~하는 법", "~절차" |
| 비교정리 | "A와 B 차이", "비용 비교" |
| 체크리스트 | 준비물, 확인 사항 |
| 사례해설 | 구체적 상황 설명 |

## 감정평가·부동산 키워드 주의사항

이 블로그 주제는 **법률·세금·금융**과 연결된다.

- 세율, 기간, 제도, 비용을 단정하지 않는다.
- 글에 **기준일**과 **전문가 확인 필요** 안내를 넣을 수 있도록 키워드를 선정한다.
- "절세 방법", "반드시 ○○원" 같은 과장 표현 키워드는 보류한다.

## keywords.csv 기록 방법

검증 후 아래 항목을 업데이트한다.

1. `validation_status`: 후보 → 검증중 → 검증완료 (또는 보류)
2. `source_hint`: 확인한 도구 목록
3. `competition_note`: 1페이지 관찰 메모
4. `cpc_note`: 단가 가능성 질적 메모 (선택)
5. `memo`: 작성 시 주의할 점

검증이 완료된 키워드만 초안 생성 대상으로 삼는다.
