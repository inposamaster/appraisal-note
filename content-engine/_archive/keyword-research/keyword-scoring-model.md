# 키워드 스코어링 모델

여러 출처의 검증 결과를 바탕으로 **우선순위 의사결정**을 돕는 점수표이다.

**이 점수는 절대적 수치가 아니다.** 검색량·순위를 보장하지 않는다.

## 점수 항목

| 항목 | 배점 | 설명 |
|---|---|---|
| 검색 의도 명확성 | 1~5 | 첫 검색만으로 독자가 원하는 게 분명한가 |
| 광고 단가 가능성 | 1~5 | 금융·세금·부동산 연계로 CPC 가능성 (질적 판단) |
| 경쟁 약함 정도 | 1~5 | SEO 1페이지에 진입 여지 (수동 관찰) |
| 내 전문성/작성 가능성 | 1~5 | 실무적으로 쓸 수 있는 깊이가 있는가 |
| 콘텐츠 확장성 | 1~5 | 시리즈·내부링크·관련 글로 확장 가능한가 |
| 법률/세금/금융 리스크 | -1~-5 | 단정·오류 시 피해 큰 주제일수록 감점 |
| **총점** | 합계 | |

## 채점 가이드

### 검색 의도 명확성 (1~5)

| 점수 | 기준 |
|---|---|
| 5 | "~하는 법", "~차이", "~필요한 경우"처럼 질문이 하나로 고정 |
| 3 | 의도는 있으나 범위가 넓음 |
| 1 | 의미 모호, 여러 의도 혼재 |

### 광고 단가 가능성 (1~5)

- Keyword Planner·Search Ad 지표는 **참고만**
- 숫자가 아닌 "금융·상속·세금 연계 → 단가 가능성 있음" 수준으로 판단

### 경쟁 약함 정도 (1~5)

- `manual-google-related-search.md` 1페이지 관찰 기준
- 개인 블로그·Q&A 혼재 → 4~5
- 대형 사이트만 → 1~2

### 내 전문성/작성 가능성 (1~5)

- 감정평가·부동산 실무 관점을 넣을 수 있는가
- 사실 확인 부담이 감당 가능한가

### 콘텐츠 확장성 (1~5)

- 관련 글 2~3편 이상 연결 가능한가
- 예: 상속 감정평가 → 가액 산정 → 비용 → 절차

### 법률/세금/금융 리스크 (-1~-5)

| 감점 | 예시 |
|---|---|
| -1 | 일반 개념 설명 |
| -3 | 세금·비용 언급 필요 |
| -5 | 절세·회피, 구체적 세율 단정 위험 |

감정평가, 상속, 증여, 경매, 부동산 키워드는 **기본적으로 리스크 항목을 확인**한다.

## 총점 해석 (가이드)

| 총점 | 제안 |
|---|---|
| 18 이상 | 우선 작성 검토 |
| 12~17 | 다른 후보와 비교 후 결정 |
| 11 이하 | 보류 또는 재검증 |

검색량이 **작아도** 의도·단가·경쟁이 좋으면 우선순위가 높을 수 있다.  
검색량이 **크다고** 무조건 좋은 키워드가 아니다.

## 기록 위치

### keywords.csv (확장 열 적용 시)

- `score_total`: 총점
- `seo_competition_level`: 낮음 / 중간 / 높음 (질적)
- `estimated_search_demand`: 낮음 / 중간 / 높음 (질적)
- `next_action`: 초안생성 / 재검증 / 보류

### raw-data

항목별 근거는 해당 날짜의 raw-data 파일에 남긴다.
