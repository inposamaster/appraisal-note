# 아카이브

이 폴더는 **일상 작업에서 쓰지 않는** 이전 버전 문서입니다.  
삭제하지 않고 보관만 합니다. 필요할 때만 꺼내 씁니다.

## 포함 내용

| 경로 | 용도 |
|---|---|
| `keyword-research/` | Keyword Research Lab v0.1 (심화 키워드 검증) |
| `raw-data/` | Google·Naver 수동 검색 기록 (2026-07-01 조사 포함) |
| `workflow.md` | 구버전 상세 워크플로 |
| `keyword-research-guide.md` | 구버전 키워드 가이드 |
| `visual-elements-guide.md` | 시각 요소 가이드 (article-template.md로 흡수) |
| `keywords.csv` | content-plan.csv 통합 전 파일 |
| `.env.example` | API 연동 시 참고용 |
