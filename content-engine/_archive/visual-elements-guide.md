# 시각 요소 가이드

시각 요소는 장식이 아니라 **독자의 이해를 돕는 기능 부품**이다.

## 유형별 사용 기준

| 상황 | 권장 요소 |
|---|---|
| 두 개념 비교 | 표 |
| 절차·단계 설명 | 번호 목록 또는 체크리스트 |
| 핵심 결론·요약 | 요약 박스 (인용 블록 활용) |
| 복잡한 관계 | 도식·흐름도 (이미지 자리표시자) |

## 요약 박스

Hugo 마크다운에서 인용 블록(`>`)으로 핵심 요약을 표시한다.

예시 형태:

> **핵심 요약**
>
> - 첫 번째 핵심 포인트
> - 두 번째 핵심 포인트

## 비교 표

비교 개념은 표로 정리한다. 예: "시세 vs 감정평가", "갑구 vs 을구"

| 항목 | A | B |
|---|---|---|
| 정의 | | |
| 용도 | | |
| 주의점 | | |

## 체크리스트

절차형 주제는 단계별 체크리스트를 넣는다.

- [ ] 1단계: 확인 사항
- [ ] 2단계: 확인 사항
- [ ] 3단계: 확인 사항

## 이미지 자리표시자 (v0.1)

아직 실제 이미지 자동 생성은 하지 않는다. 글 안에는 **자리표시자와 alt/caption 초안**만 넣는다.

마크다운 형태:

    ![alt 텍스트 초안](images/파일명-예정.png)
    *캡션: 이미지 설명 초안*

### 이미지 규칙

- 저작권 문제가 없는 것만 사용한다.
- 구글 이미지 검색 결과를 임의로 가져오지 않는다.
- 가능하면 직접 만든 도식, 흐름도, 표 이미지를 사용한다.
- `alt` 텍스트와 캡션을 반드시 붙인다.
- 웹에 맞게 용량을 줄인다 (권장: 200KB 이하).

### 저장 위치

- 이미지 파일: `static/images/` 또는 `content/posts/` 인접 폴더
- v0.1에서는 자리표시자만 작성하고, 실제 파일은 검수 단계에서 추가한다.

## 검수 시 확인

- 시각 요소 없이 긴 문단만 이어지지 않는가
- 표·박스가 본문 내용과 일치하는가
- 이미지 alt가 검색·접근성에 도움이 되는가
